- Chủ đề: Thành hành chức năng PWM của Timer trong STM32
- Chức năng: Tăng độ sáng đèn led sau đó tắt rồi tăng độ sáng léd, lặp lại.
- Cách sử dụng:
1. Mở file cấu hình (.ioc) để xem cấu hình STM32.
2. Gắn dây nạp vào STM32.
3. Mở file project trên STM32CubeIDE để compile và chạy chương trình.
4. Quan sát led có sẵn trên board để thay sự thay đổi độ sáng của led.
